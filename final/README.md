How to Run It

Open the Xcode project, run it on the iOS Simulator or a physical device, and navigate through the home screens to explore recipes, tools, and timers.

⸻

Features
	•	Browse or add custom recipes
	•	Built-in cooking timers and multi-step timing
	•	Pantry tracking for ingredients
	•	Order-tracking flow (for baking or selling goods)
	•	Tools like voice-to-text conversions and measurement converters

⸻

Screenshots

Screens such as:
	•	Home / Recipes page
	•	Add Recipe page
	•	Tools page
	•	Timer creation page
	•	Pantry and Add Item screens
	•	Order tracking and New Order form

⸻

Setup Instructions
	1.	Clone or download the project folder.
	2.	Open the .xcodeproj in Xcode.
	3.	Ensure you are running the latest iOS SDK.
	4.	Build and run the project using the iOS Simulator or a connected device.

⸻

Known Issues
	•	Voice-to-text accuracy may vary depending on background noise.
	•	Some UI elements may need optimization for smaller iPhone screens.
	•	Timers do not send notifications when the app is fully closed.
